Send automated WhatsApp messages to customers and admins based on SureCart order lifecycle events using the Meta WhatsApp Cloud API.

== Description ==

SureCart WhatsApp Notifications sends automated WhatsApp messages to your customers and admin team when key order events happen in SureCart:

* **Order Confirmed** - Notify customers when their checkout is confirmed
* **Fulfillment Created** - Notify customers when their order is shipped
* **Refund Issued** - Notify customers when a refund is processed
* **Admin Alert** - Notify admins when a new order comes in

Features:

* Easy setup with Meta WhatsApp Cloud API
* Customizable message templates with variable substitution
* Support for Meta pre-approved templates
* Activity dashboard with stats and charts
* Detailed notification logs with filtering
* Retry failed notifications
* E.164 phone number normalization
* Encrypted API credential storage

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate through the 'Plugins' menu in WordPress
3. Go to WhatsApp > Settings to configure your Meta Cloud API credentials
4. Set up your message templates and notification preferences
